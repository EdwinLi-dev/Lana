全称 **Library with auto asking and answer**<br/>
本项目基于 Springboot + JPA + Mybatis。核心功能使用 **知识图谱** (Knowledge Graph) <br/>
单点数据库 MySQL.
## 使用说明
本项目为 Maven 构建的 Springboot 项目。克隆后直接构建、打包即可启动。
